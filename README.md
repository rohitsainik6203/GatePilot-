# GatePilot AI

An AI-powered GATE preparation dashboard designed to help students plan, track, and optimize their exam preparation.

---

## Features

- Smart study planner
- GATE exam countdown
- Subject progress tracking
- Revision heatmap
- Weak topic detection
- Daily study targets
- Progress analytics
- AI-based recommendations
- Dark mode dashboard
